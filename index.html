How it works

Buildings of the same type share a speed upgrade. The fed line shows how much of each input it's getting per second vs. how much it needs at full speed — green means kept up, red means starved. The running bar shows how fast it's actually working out of its max. You pick where each output routes — into a downstream building or a Shop, which only sells so fast. Place multiples to keep up: one Wheel makes 2/s but an Assembler needs 4/s.
